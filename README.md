# 📸 Photo Studio — Free Online Image Editor

> **Ek powerful, mobile-friendly photo editor — koi installation nahi, koi login nahi, bilkul free!**

🌐 **Live Demo:** [tusharsharmalogic-cmyk.github.io/Photo-editor](https://tusharsharmalogic-cmyk.github.io/Photo-editor/)

---

## ✨ Features

### 🔧 Transform Tools
- **Rotate** — Left & Right 90° rotation
- **Flip** — Horizontal & Vertical mirror
- **Crop** — Free crop ya fixed ratio (1:1, 16:9, 9:16, 4:3, 3:2, aur bahut kuch)
- **Sharpen** — Photo ko crisp aur sharp banao

### 🎛️ Real-Time Adjustments
- **Brightness** — Roshni badhao ya ghataao
- **Contrast** — Colors ko aur vivid banao
- **Saturation** — Color intensity control
- **Blur** — Background blur effect
- **Opacity** — Transparency control
- **Hue Rotate** — Puri color tone badlo

### 🎨 One-Click Filters
- 🌫️ **Grayscale** — Black & White classic look
- 🟫 **Sepia** — Vintage warm tone
- 🔄 **Invert** — Negative effect
- 🔍 **Sharpen** — Detail enhance karo

### 📐 Smart Resize
- Custom width & height set karo
- **Aspect Ratio Lock** — Photo stretch nahi hogi
- **Quick Presets:**

| Preset | Size | Use Case |
|--------|------|----------|
| Full HD | 1920×1080 | Wallpaper / YouTube |
| 720p | 1280×720 | Video Thumbnail |
| Instagram | 1080×1080 | Square Post |
| Story | 1080×1920 | Instagram / WhatsApp Story |
| Facebook | 1200×628 | Cover / Post |
| Avatar | 400×400 | Profile Picture |
| 2K | 2560×1440 | High-res Display |

### 💾 Export Options
- **Formats:** PNG · JPEG · WebP
- **Quality Control** — File size vs quality balance
- **Custom File Name** — Apna naam do download se pehle

### 📱 Mobile Friendly
- Touch support — Crop finger se drag karke karo
- Tab-based navigation — Sab tools ek jagah, screen waste nahi
- Kisi bhi phone browser mein khulta hai

---

## 🚀 How to Use

```
1️⃣  Upload Tab  →  Photo choose karo (ya drag & drop)
2️⃣  Tools Tab   →  Rotate, Flip, Filters lagao
3️⃣  Adjust Tab  →  Brightness, Contrast, etc. adjust karo
4️⃣  Crop Tab    →  Ratio choose karo → Crop Mode → Drag karo
5️⃣  Resize Tab  →  Size change karo ya preset chunao
6️⃣  Save Tab    →  Format & quality chunao → Download!
```
